Bài Tập 1:
Mô tả bài tập:
- Hiển thị dữ liệu được từ API có sẵn và hiển thị trên trang web
- Xóa dữ liệu trên API và tải lại API đó, vẫn lưu lại thay đổi.
Hướng dẫn thao tác:
- Khi click vào button Search, trang web sẽ chạy API có sẵn được cung cấp ở trong Function
- Hiển thị kết quả trả về theo dạng JSON. Từ đó hiển thị các dữ liệu đó trên trang web
- Có thể xóa dữ liệu của kết quả trả về bằng cách click vào dữ liệu cần xóa. Sau tầm 4-5s, dữ liệu sẽ bị xóa khỏi dãy JSON
và dãy JSON sẽ lưu lại những thay đổi đó. (Có thể refresh trang web và click vào button search)
