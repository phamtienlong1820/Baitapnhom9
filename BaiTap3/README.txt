Mô tả Bài Tập 3:
-Sử dụng Bootstrap 
-Tạo trang web index.html có các yêu cầu sau:
+ Thanh menu: chứa các link home, about, product, register và form search
+ Silde trình chiếu
+ Các sản phẩm: chứa các thông tin của 1 sản phẩm: Tên, hình, giá, 2 button: mua và xem chi tiết
+ Footer
+ Trang web phải thay đổi kích thước theo thiết bị (responsive)
+ Khi click vào Register, hiện modal chứa form nhập thông tin đăng ký
+ Mỗi sản phẩm đặt trong 1 div.card

Hướng dẫn cài đặt:
- Đặt folder BaiTapNhom9 vào folder htdocs của Folder Xampp
- Sử dụng Web Browser và ghi url:localhost/BaiTapNhom9 
- Chọn đường link Bài Tập 3

Hướng dẫn thao tác:
- Silde trình chiếu banner có thể bấm tới hoặc lui về (Tự động qua silde sau 3 giấy).
- Trang web thay đổi theo kích thước thiết bị sử dụng (Nhấn F12 và chọn mode Toggle device emulation, sau đó thay đổi kích thước mình muốn)
- Khi click vào đường link Register trên thanh menu, Modal form thông tin đăng ký sẽ xuất hiện trên trang web
